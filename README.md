## How to Use

1. Open `index.html` in any web browser
2. The chatbot will automatically start with a welcome message
3. Click on any menu option to explore
4. Use "Back to Main Menu" button to return to main options
5. Navigate freely between different sections

## Customization

To update account details, edit the `cashGift` object in `script.js`:

```javascript
cashGift: {
    accountName: "Your Account Name",
    accountNumber: "Your Account Number",
    bankName: "Your Bank Name"
}
```
